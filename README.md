# Nuxt 3 全栈开发
使用Nuxt开发的个人Blog程序

## 技术栈

Nuxt3，Pinia，Element-plus，Tailwindcss，Mysql，Prisma，JWT...

## 起步

下载好源码后，执行命令安装依赖包。

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## 开发服务

开发服务端口 `http://localhost:9080`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```

## 打包

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build

# bun
bun run build
```
